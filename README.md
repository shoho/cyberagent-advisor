# cyberagent-advisor

サイバーエージェントの経営哲学・組織論・人材育成論に基づくアドバイスを提供する [Claude Code](https://docs.anthropic.com/en/docs/claude-code) のスキルです。

## 概要

「サイバーエージェントだったらどうするか？」という視点で、事業・経営・組織・人材育成・採用・カルチャー・技術戦略に関する具体的かつ実践的なアドバイスを提供します。

藤田晋、曽山哲人をはじめとするキーマンの考え方や、あした会議・CAKK制度・新卒社長制度・抜擢メソッドなどの具体的な制度・施策を参照しながら回答します。

## インストール

```bash
claude skill add --from shoho/cyberagent-advisor
```

## 使い方

Claude Code 上で `/cyberagent-advisor` と入力するか、サイバーエージェントに関連する質問をすると自動的にスキルが起動します。

### トリガーとなるキーワード例

- 「サイバーエージェントだったら」「CAだったら」
- 「サイバー流」「藤田さんなら」「曽山さんなら」

### 質問カテゴリ

- **経営判断** - 意思決定フレームワーク、長期投資判断
- **組織づくり** - 実力主義型終身雇用、挑戦と安心のバランス
- **人材育成** - 抜擢メソッド、新卒社長制度、後継者育成
- **新規事業** - あした会議、撤退基準（CAKK制度）、ABEMAの事例
- **採用** - 「素直でいいやつ」基準、ポテンシャル採用
- **カルチャー** - ミッションステートメント、maxims
- **技術戦略** - AI戦略、エンジニア評価制度

### 質問例

```
サイバーエージェントだったら、新規事業の撤退判断はどうする？

うちの会社で若手の抜擢を進めたいが、どんな制度設計がいい？

エンジニア組織の評価制度を見直したい。CAではどうしている？
```

## ファイル構成

```
.
├── SKILL.md                          # スキル定義（プロンプト・回答手順）
└── references/
    ├── knowledge-base.md             # ナレッジベース（経営哲学・制度・人物情報）
    └── .claude/
        └── settings.local.json       # ローカル設定
```

## ナレッジベースの内容

- 企業概要と歴史（1998年創業〜28期連続増収〜2025年社長交代）
- 経営哲学（ビジョン、パーパス、21世紀型の日本的経営）
- ミッションステートメント（15項目）/ maxims（8つの価値観）
- 組織づくりの原則（実力主義型終身雇用、CA18制度等）
- 人材育成の方法論（抜擢メソッド、才能開花の3要素、新卒社長制度）
- 意思決定フレームワーク（主観・客観・俯瞰、撤退基準）
- 新規事業創出の仕組み（あした会議、ジギョつく、ABEMA事例）
- 採用の考え方（「素直でいいやつ」基準）
- 技術組織・AI戦略（JBキャリアプログラム、AI Lab、OpenCALM）
- キーマンの考え方と名言（藤田晋、曽山哲人）
- 具体的制度・施策一覧（30+制度）
- 役員・執行役員・事業責任者のプロフィール
- 書籍リスト
