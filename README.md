# cyberagent-advisor

サイバーエージェントの経営哲学・組織論・人材育成論に基づくアドバイスを提供する [Claude Code](https://docs.anthropic.com/en/docs/claude-code) のスキルです。

## 概要

「サイバーエージェントだったらどうするか？」という視点で、事業・経営・組織・人材育成・採用・カルチャー・技術戦略に関する具体的かつ実践的なアドバイスを提供します。

回答時には必ずウェブ検索で最新情報を収集し、ナレッジベースと統合して回答します。藤田晋（会長）、山内隆裕（社長）、曽山哲人（CHO）をはじめとする経営陣の考え方や、あした会議・CAKK制度・新卒社長制度・抜擢メソッドなど30以上の具体的な制度・施策を参照しながら回答します。

## インストール

```bash
claude skill add --from shoho/cyberagent-advisor
```

## 使い方

Claude Code 上で `/cyberagent-advisor` と入力するか、サイバーエージェントに関連する質問をすると自動的にスキルが起動します。

### トリガーとなるキーワード例

- 「サイバーエージェントだったら」「CAだったら」
- 「サイバー流」「藤田さんなら」
- 事業・経営・組織・人材育成・採用・カルチャー・技術戦略に関する質問

### 質問カテゴリ

- **経営判断** - 意思決定フレームワーク（主観・客観・俯瞰）、長期投資判断、押し引きの見極め
- **組織づくり** - 実力主義型終身雇用、挑戦と安心のバランス、CA18制度による役員新陳代謝
- **人材育成** - 抜擢メソッド、新卒社長制度、後継者育成（CE研修）、才能開花の3要素
- **新規事業** - あした会議、ジギョつく、撤退基準（CAKK制度）、ABEMAの長期投資事例
- **採用** - 「素直でいいやつ」基準、ポテンシャル採用
- **カルチャー** - ミッションステートメント（15項目）、maxims（8つの価値観）
- **技術・エンジニアリング** - AI戦略（AI Lab, OpenCALM, AIドリブン推進室）、JBキャリアプログラム
- **事業戦略** - 事業ポートフォリオ管理、ABEMA黒字化の軌跡

### 質問例

```
サイバーエージェントだったら、新規事業の撤退判断はどうする？

うちの会社で若手の抜擢を進めたいが、どんな制度設計がいい？

エンジニア組織の評価制度を見直したい。CAではどうしている？

藤田さんが社長を退任した理由と後継者選びのポイントは？

AI活用を全社で推進したい。CAのAI戦略を参考にしたい。
```

## 回答の特徴

- 回答前に必ずウェブ検索で最新情報を収集（CyberAgent Way、藤田晋ブログ等の一次情報を優先）
- ナレッジベースの体系的知識と最新情報を統合して回答
- 抽象論だけでなく「あなたの場合はこう応用できる」まで踏み込む
- 経営判断には10項目の判断チェックリストを適用

## ナレッジベースの内容

1. **企業概要と歴史** - 1998年創業〜28期連続増収〜2025年12月社長交代（山内隆裕新社長就任）
2. **経営哲学** - ビジョン「21世紀を代表する会社を創る」、パーパス、21世紀型の日本的経営
3. **ミッションステートメント** - 行動規範15項目
4. **maxims** - 社員全員に配布される8つの価値観
5. **組織づくりの原則** - 実力主義型終身雇用、挑戦と安心のセット設計、CA18制度
6. **人材育成の方法論** - 抜擢メソッド（自走サイクル）、才能開花の3要素、新卒社長制度（延べ50人以上）、CE研修（後継者育成）
7. **意思決定フレームワーク** - 主観・客観・俯瞰の3視点、CAKK制度（撤退基準）、押し引きの見極め
8. **新規事業創出の仕組み** - あした会議（累計売上700億円）、ジギョつく、ABEMA長期投資事例（2024年黒字化達成）
9. **採用の考え方** - 「素直でいいやつ」基準、ポテンシャル採用
10. **技術組織・AI戦略** - JBキャリアプログラム、AI Lab、OpenCALM、AIドリブン推進室（2025年新設）、全社AIリスキリング
11. **キーマンの考え方と名言** - 藤田晋（創業者・会長）、その他役員・執行役員
12. **具体的制度・施策一覧** - 30以上の制度（経営・事業、新規事業、採用、育成、評価、適材適所、活性化、福利厚生、技術）
13. **役員・執行役員・事業責任者** - 取締役7名、執行役員体制（副社長〜常務16名＋執行役員17名）、主要人物プロフィール、グループ会社経営者
14. **書籍リスト** - 藤田晋・曽山哲人の著書
